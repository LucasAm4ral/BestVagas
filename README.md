# BestVagas
 Site de vagas para desafio PHP


Intruções: 

-Subir o script do banco de dados no localhost

-Baixar o projeto na pasta htdocs do xampp

-Para utilizar o site basta acessar http://localhost/BestVagas/index.html

-Caso tenha duvidas sobre as funcionalidades do sistema, acesse a aba na barra de navegação "Como utilizar o Site"
nela ira conter instruções de como operar cada funcionalidade dos sistema.   

Obrigado por visualizar o projeto! =)
